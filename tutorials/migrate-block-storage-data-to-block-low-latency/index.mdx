This tutorial shows you how to migrate your data from Block Storage volumes (b_ssd) to Block Storage Low Latency volumes.
Note that your Operating System and the volume on which you have stored it, cannot be migrated to Block Storage Low Latency, as the boot on block feature is not available for Block Storage Low Latency.

<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have [created an SSH key](/console/project/how-to/create-ssh-key/)
  - You have [an Instance](/compute/instances/how-to/create-an-instance/) with data stored on a Block Storage `b_ssd` volume
</Message>
