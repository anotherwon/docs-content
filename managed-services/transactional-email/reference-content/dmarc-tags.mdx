This documentation provides information about the tags you can configure when [adding a DMARC record](/managed-services/transactional-email/how-to/add-dmarc-record/) to your domain.



| Tag   | Description                                                                             | Mandatory | Values expected              | Default |
|-------|-----------------------------------------------------------------------------------------|-----------|------------------------------|---------|
| v     | Indicates the DMARC version                                                             | Yes       | DMARC1                       |    x    |
| p     | Indicates the policy to apply when/if a check fails                                     | Yes       | - none - quarantine - reject | x       |
| sp    | Indicates the subdomain policy to apply when/if a check fails                           | No        | - none - quarantine - reject | x       |
| adkim | Defines the DKIM alignment mode                                                         | No        | - r (relaxed) - s (strict)   | r       |
| aspf  | Defines the SPF alignment mode                                                          | No        | - r (relaxed) - s (strict)   | r       |
| pct   | Percentage of messages from the domain owner's mail stream to apply the DMARC policy to | No        | 0 - 100                      | 100     |

| rua   |                                                                                         |           |                              |         |
